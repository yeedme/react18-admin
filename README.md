💻︎<a href="https://yeedme.github.io/login">在线预览</a>


如何使用
- 获取项目代码

```bash
git clone https://github.com/yeedme/react18-admin.git
```

 - 切换目录

```bash
cd react18-admin
```
- 项目初始化
```bash
npm i 
```
- 启动项目
```bash
npm run dev
```
